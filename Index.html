Loja Online

Descrição do Projeto

Este projeto é uma aplicação web completa de e-commerce, simulando o funcionamento real de uma loja virtual.

A plataforma conta com uma tela de login segura, uma página inicial com carrossel de destaques, uma área de produtos organizada por categorias, carrinho de compras dinâmico e processo de finalização de pedido. No painel administrativo, o sistema permite cadastro e edição de produtos, controle de estoque em tempo real, visualização de relatórios de vendas e gerenciamento de pedidos. A interface é responsiva e adaptada para diferentes dispositivos, com foco em praticidade e usabilidade tanto para o cliente quanto para o administrador.

Tecnologias utilizadas:
HTML5, CSS3, JavaScript, PHP, Bootstrap e MySQL.
