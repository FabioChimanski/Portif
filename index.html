Loja Online

Descrição do Projeto

Este projeto é uma aplicação web completa de e-commerce, simulando o funcionamento real de uma loja virtual.

A plataforma conta com uma tela de login segura, página inicial com carrossel, produtos com categorias, carrinho dinâmico e painel administrativo para cadastro de produtos, controle de estoque e relatórios de vendas.

Tecnologias utilizadas:
HTML5, CSS3, JavaScript, PHP, Bootstrap e MySQL.

Consultório

Descrição do Projeto

Este projeto é um sistema web para organização de atendimentos em consultórios médicos, focado na comunicação entre recepção e pacientes.

O sistema permite cadastrar pacientes e realizar chamadas por parte do médico. Essas chamadas são exibidas em tempo real em uma tela de recepção, informando o nome do paciente e o consultório correspondente.

Tecnologias utilizadas:
HTML5, CSS3, JavaScript e Bootstrap.
